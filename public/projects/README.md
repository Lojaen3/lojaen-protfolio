# Project screenshots

Add your images here. Vite serves files in `public/` at the **site root**, so  
`/projects/unitok.jpg` → this folder, file `unitok.jpg`.

## Filenames (match `ProjectsSection.jsx`)

| File | Project |
|------|---------|
| `unitok.jpg` | UniTok |
| `rewaqx.jpg` | Rewaqx |
| `righthand.jpg` | RightHand |
| `planto.jpg` | Planto |
| `cyclash.jpg` | Cyclash |
| `borikat.jpg` | Borikat |
| `travel-roommate.jpg` | Travel Roommate |

**Formats:** `.jpg`, `.jpeg`, `.png`, or `.webp` — if you use a different name or extension, update the `image:` path for that project in `src/components/portfolio/ProjectsSection.jsx`.

**Tips**

- Use **16:9** or **4:3** screenshots; cards use a fixed height and `object-cover`.
- Prefer **compressed** images (e.g. WebP) for faster loading.
- If a file is missing, the site falls back to the placeholder URLs from `Home.jsx`.
